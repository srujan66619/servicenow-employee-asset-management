# Employee Asset Management - ServiceNow

## Overview
A ServiceNow application for managing employee asset requests.

## Features
- Custom Asset Request table
- Reference and Choice fields
- Flow Designer automation
- Email notifications
- Reports and dashboards
- ACL-based security

## Workflow
Employee Request → Manager Approval → IT Assignment

## Technologies
- ServiceNow Studio
- Flow Designer
- Notifications
- Reports
- ACLs
